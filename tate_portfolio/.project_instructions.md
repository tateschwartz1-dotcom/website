# Tate Schwartz Portfolio

## Design System
- **Fonts**: "Press Start 2P" (pixelated headers), "Courier Prime" (body/links)
- **Colors**: Coral (#F5A88E), Mint (#B4E7CE), Lavender (#D4C5F9), Peach (#FFD4B2), Sky (#C5E3F6). Text: Charcoal (#3A3A3A)
- **Theme**: Toggle cycles through colors. Button fixed bottom-left.

## Pages
- `/` — Homepage with nav, social links, news ticker, wave easter egg
- `/photo` — Polaroid photo cycler. Header changes to "Photos of me" on click.
- `/projects` — 3-column grid. Detail pages at `/projects/[id]`
- `/favorite-things` — Bulleted list with gradient scroll + many easter eggs
- `/beliefs` — Draggable blocks. Hidden "Boo" behind "Shooters shoot"
- `/golf` — Mini golf game (in development). 9 holes planned, each themed to Tate.

## Golf Game Architecture
- Engine: Canvas-based 2D mini golf in `components/golf/golf-canvas.tsx`
- Types: `lib/golf-types.ts` — Vec2, Wall, RectObstacle, CircleObstacle, HoleConfig, GameState
- Holes: `lib/golf-holes.ts` — hole configurations
- Physics: velocity, friction (0.985), wall reflection with energy loss (0.75), circle obstacle bouncing
- Aiming: Click near ball + drag to set direction/power. Slingshot style (pull back).
- Max power: 18, power scale: 0.12, ball radius: 8, hole sink speed check
- Page at `/golf` with stroke counter, par display, completion overlay
- Hole not yet linked from homepage nav (to be added when 9 holes ready)
